宏观经济（二）——货币政策与财政政策

⏱ 约 10 分钟  ·  货币本质 · 央行 · M2 · 信用
💡 一句话总结

央行是市场的"天气管理局"——它决定刮什么风向。聪明的投资者会在出门前先看天气预报。

🔑 核心知识
01 货币政策（央行说了算）： 中国人民银行（PBOC）的主要工具： 存款准备金率（RRR）：商业银行必须在央行存留的存款比例 ↓降准 → 银行可贷资金增多 → 市场流动性充裕 → 利好股市/楼市 ↑升准 → 收缩流动性 → 利空 MLF（中期借贷便利）& LPR：影响银行的资金成本 → 进而影响贷款利率 公开市场操作（OMO）：逆回购/正回购 → 调节短期流动性 再贷款/再贴现：定向支持特定领域 美联储（Fed）的政策工具： 联邦基金利率（Federal Funds Rate）→ 全球资产的定价锚 QE（量化宽松）/ QT（量化紧缩） 加息/降息对全球资本的"虹吸效应" "宽货币" vs "紧货币"： 宽货币：央行放水 → 利率下行 → 资产价格上涨（利好股/债/房） 紧货币：央行收水 → 利率上行 → 资产价格承压 财政政策（政府说了算）： 减税降费 → 企业和个人手头钱多了 → 刺激消费投资 增加支出（基建投资/补贴）→ 直接拉动需求和就业 发行特别国债 → 为重大项目建设筹资 加税 → 抑制 overheated 的经济 特别国债/专项债：中国特有的财政工具，用于支持重大项目建设 "货币+财政"组合拳的效果： 宽货币 + 宽财政 = 强刺激（如2020年疫情期间） 紧货币 + 紧财政 = 强紧缩（较少见） 宽货币 + 紧财政 = 结构性分化（资金多但政府不花钱） 紧货币 + 宽财政 = "挤出效应"（政府借钱推高利率，私人部门被挤出）
02 美国：从激进加息转向逐步降息（对抗高通胀取得成效后的政策转向） 影响：中美利差变化 → 人民币汇率压力 → A股外资流向 → QDII收益
03 中国：货币政策保持稳健偏宽松（支持实体经济），财政政策发力（特别国债/专项债/设备更新以旧换新补贴等）
