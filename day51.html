基本面分析（二）——关键财务指标深度解读

⏱ 约 15 分钟  ·  财务报表 · ROE · PE · 估值
💡 一句话总结

财务指标不是孤立的——要把多个指标放在一起看，结合行业特点和公司历史趋势来判断。单一指标很容易误导人。

🔑 核心知识
01 毛利率 = (营收 - 成本) / 营收 >50% = 极强竞争力（如茅台90%+/医药/软件） 20-40% = 有一定优势 <20% = 竞争激烈/成本压力大 净利率 = 净利润 / 营收（>10%算健康） 成长性指标（公司还在变好吗）： 营收同比增长率（>15%算高增长） 净利润同比增长率（>20%算高增长） 连续3年以上增长 = 成长性好 ⚠️ 要区分"内生增长"（业务本身在增长）和"外延增长"（靠并购堆出来的） 偿债能力指标（公司会不会突然倒闭）： 资产负债率 = 总负债 / 总资产 <40% = 非常保守 40-60% = 健康（大多数公司在此区间） 60-70% = 偏高需关注 流动比率 = 流动资产 / 流动负债（>1.5算安全） 速动比率 = (流动资产-存货) / 流动负债（剔除存货后>1算安全） 利息保障倍数 = EBIT（息税前利润） / 利息支出（>3倍才算安全） 运营效率指标（公司管理得好不好）： 存货周转率 = 营业成本 / 平均存货（越高说明卖得越快） 应收账款周转天数 = 365 / (营收/平均应收账款)（越短说明回款越快） 总资产周转率 = 营业收入 / 平均总资产 现金流指标（利润是不是"真金白银"）： 经营现金流/净利润 > 1 → 利润质量好（赚的都是真钱） 经营现金流/净利润 < 1 甚至为负 → ⚠️ 警惕！利润可能是纸上富贵 自由现金流（FCF）= 经营现金流 - 资本开支（公司真正可以自由支配的现金）
02 巴菲特最看重的指标！>15%算优秀，>20%算卓越，<8%算平庸 杜邦分解：ROE = 净利率 × 资产周转率 × 权益乘数
03 通过杜邦分解可以看出ROE的来源是高利润率（如茅台）还是高周转（如沃尔玛）还是高杠杆（如银行）
04 ROE（净资产收益率 Return on Equity）= 净利润 / 平均净资产
